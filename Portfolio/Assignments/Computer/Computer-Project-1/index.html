Introduction:

Devices:

The components which are attached to the computer to enter the data and get desired result are called as "Devices".

Input And Output Devices:

An input device is a piece of equipment used to provide data in computing.Input devices are hardwares use to enter data for processing. An output device is a piece of computer hardware which converts information into human-readable form.It can be text, graphics,audio and video.
                                                                   OR
We can say that:An output device is a hardware device used to send data from a computer to another device or user.
